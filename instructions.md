## Create bootable macOSX 10.12 Sierra install on 8GB USB stick, from install file on desktop
sudo ~/Desktop/Install\ macOS\ Sierra.app/Contents/Resources/createinstallmedia --volume /Volumes/Install\ macOS\ Sierra --applicationpath ~/Desktop/Install\ macOS\ Sierra.app
